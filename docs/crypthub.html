How export works:
Downloads a .crypthub file containing all your passwords encrypted with AES-256-GCM. The file is protected by your master password — it cannot be read without it.
Keep this file somewhere safe. Anyone with this file AND your master password can restore your vault.
How import works:
Select a .crypthub backup file, enter your master password to verify, and CryptHub will restore your vault. This will completely replace your current data.
Your current vault will be permanently erased and replaced with the imported data.
